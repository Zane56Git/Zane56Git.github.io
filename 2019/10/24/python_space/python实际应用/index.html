线性回归

数学模型
一元线性回归公式

以下是公式
y=ax+b+εy=ax+b+ε
y 为应变量 dependent variable
x 为自变量 independent variable
a 为斜率 coeffient
b 为截距 intercept
ε （读作epsilon）为误差，正态分布
线性回归的目标是，找到一组a和b，使得ε最小
y^=ax+by^=ax+b
ε=y−y^ε=y−y^
y^y^ 读作y hat，也有人读作y帽子。这里的帽子一般表示估计值，用来区别真实值y。
