Linux-机器基础搭建

发表于 2018-05-17 更新于 2024-05-24 分类于 Linux Changyan：
  • 以下主要基于虚拟机快速完成基础搭建

建议使用桥接网络-实现局域网互通

固定IP，设置主机名，这是服务器的基本设置

root->@2020

创建deploy用户（后面的基本操作都必须基于非root用户完成，这是规范、标准）

1、快速虚拟化三台centos7（半个小时内），固定好IP，网络互通即可（离开虚拟机，通过ssh操作）(相当拥有了3台服务器，省去购买阿里云的钱)

2、设置ssh密钥（便于部署）

3、设置软链

sudo ln -s /opt/data/ /data
deploy用户下创建/opt/data/目录

规范

主机名：host_app1（例如：vm_host1、vm_cdh1、host_service1、server1.bigdata.local、master.cdh、node1.cdh）
host映射：server1.bigdata.local（例如：server1.bigdata.local、local.service1.com）

相关命令

https://www.cnblogs.com/zhangyy3/p/12454728.html

注意:第3点后,创建分区后需要重启再对/dev/sda3进行格式化,不重启导致格式化失败

mkfs.xfs /dev/sda3
