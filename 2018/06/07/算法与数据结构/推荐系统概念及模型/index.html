1、发现兴趣相似的用户

由上计算出用户之间的相似度

通常用 Jaccard 公式或者余弦相似度计算两个用户之间的相似度。设 N(u) 为用户 u 喜欢的物品集合，N(v) 为用户 v 喜欢的物品集合，那么 u 和 v 的相似度是多少呢：

  • Jaccard 公式

$$
x = \frac{\left | N{(u)}\cap N {(v)}\right |}{\left | N{(u)}\cup N{(v)} \right |}
$$

  • 余弦相似度

$$
x = \frac{\left | N{(u)}\cap N {(v)}\right |}{\sqrt{\left | N{(u)}| \ast |N{(v)} \right |}}
$$

https://www.jianshu.com/p/a894ebba4a1a

由Jaccard计算结果如下：

U0001 U0002 U0003
U0001 0 1/5 2/4
U0002 1/5 0 1/3
U0003 2/4 1/3 0

由余弦计算结果如下：

U0001 U0002 U0003
U0001 0 1/根号8 = 0.35355 2/根号8 = 0.70711
U0002 1/根号8 = 0.35355 0 1/根号4 = 0.5
U0003 2/根号8 = 0.70711 1/根号4 = 0.5 0

可看出：

与U0001比较相似的用户是U0003，

与U0003比较相似的用户是U0001，可以把U0001中的商品D0001、D0005推荐给U0003；

与U0002比较相似的用户是U0003，可以把U0003中的商品D0002推荐给U0002；
