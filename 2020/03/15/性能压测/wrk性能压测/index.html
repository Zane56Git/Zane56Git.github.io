wrk性能压测

发表于 2020-03-15 更新于 2023-06-15 分类于 性能压测 Changyan：

如果你想成为一名高级开发工程师，那你一定要具备压测及优化能力。常用的压测工具有：Httpd、Jmeter、Wrk、Postman、ab、http_load、webbench、siege。下面主要记录一下通过Jmeter压测记录

http://durcframework.gitee.io/sop/#/?t=1582014833549
https://www.cnblogs.com/quanxiaoha/p/10661650.html

CentOS / RedHat / Fedora

依次执行如下命令：
sudo yum groupinstall ‘Development Tools’
sudo yum install -y openssl-devel git
git clone https://github.com/wg/wrk.git wrk
cd wrk
make

将可执行文件移动到 /usr/local/bin 位置

sudo cp wrk /usr/local/bin

线程数为 8，模拟 200 个并发请求，持续 30 秒
wrk -t8 -c200 -d30s “http://10.1.31.227:8081/rest/story-service/food/getFoodById?id=2"

wrk -t12 -c400 -d30s –latency http://www.baidu.com
